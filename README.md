## 发票管理

主要是方法论，好的习惯可以精简SOP，

整理常用平台，如美团、滴滴、微信/邮箱 --> 定期整理到同一文件夹内 --> 报销完及时清理（关键一步，无需多次上传，长按多选即可，形成逻辑闭环）
